You can make a site like this with just HTML and CSS. If you upload the .html and .CSS files to Github it will host the website for you on GitHub Pages; you can even alias the domain name to make it look more professional.

This site is constructed of a series of .html files for the different pages which containt the actual content that you are seeing (like this paragraph) and a CSS file which handles all the styling. By styling I mean everything from the fonts to the position and colours of elements. You can even turn off the CSS if you wish to see what the raw .html files look like. To make the styling of the css easier, I've used a framework called Bootstrap which includes many inbuilt classes that respond to your screen size (e.g. whether you are using a mobile or a desktop).

There are lots of online resources that show how to do this such as 'Learn to Code HTML and CSS' by Shay Howe. The process is relatively straightforward and with about 10 hours work you should be able to create a site like this and understand how it works. If you want responsive design as well, you should google 'Bootstrap 5 tutorial' or similar. To learn, I used a tutorial which I subsequently found out was for an older version of Bootstrap so I won't recommend it here. Alternatively, you could just clone my repository on GitHub and change the content. This should only take 1 hour.
